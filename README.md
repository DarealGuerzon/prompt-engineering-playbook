AI Prompt Toolkit

A curated set of AI prompts for full-stack development, automation, analytics, and security tasks.  
These are the **top prompts I use 80% of the time**.

---

## Top Prompts

### 1. Codebase Understanding
I want you to act as a senior full-stack engineer. Help me understand this [WordPress / Node.js / Laravel / React / Vue] project thoroughly. Explain architecture, folder structure, major components, data flow, and key dependencies. Suggest areas to improve modularity, performance, or maintainability.



### 2. Adding / Modifying Features
I want to add [new feature] to this [WordPress / React / Node.js / Laravel] project. Provide step-by-step instructions, including which files to modify, how to integrate it safely, and how to maintain existing functionality. Include reusable code snippets and best practices.



### 3. Security Review
Perform a security review of this project. Identify vulnerabilities like broken authentication, insecure APIs, XSS/SQL injection, or improper data handling. Suggest fixes and best practices to harden the codebase.



### 4. Analytics / Tracking
Review the GA4 / GTM / Meta Pixel / TikTok Pixel / LinkedIn integration in this project. Identify missing events, broken triggers, or duplicate tracking issues. Suggest improvements for accuracy and reliability with step-by-step instructions.



### 5. Workflow Automation
I want to automate repetitive tasks in this [Python / Node.js] workflow. Suggest scripts, scheduling, or CI/CD integration. Provide code snippets, instructions for testing, and error handling. Explain how this reduces manual workload without disrupting existing functionality.
